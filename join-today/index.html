Welcome to the official Maine Maritime Academy Minecraft server website! We have a vanilla and a modded server here at the academy. Quick guides to joining:

 

Vanilla

  1. Launch Minecraft in version 1.21.11 (click here for a guide on changing versions)
  2. Go to multiplayer tab and click "Add Server" at the bottom
  3. Name the server whatever you like, and insert this in the IP address bar:

vanilla.mmaminecraft.org

You can join the server now!

Modded

  1. Install Curseforge and link it to your Minecraft account. You can install the standalone version, Overwolf is not required. (click here for a guide)
  2. Look up MMA Craft on Curseforge. Install the modpack uploaded by MaineMinecraft.
  3. Make sure the Curseforge RAM settings are appropriate for your computer.
    • Guides on checking RAM amount for Windows 11 and Apple
    • Change the RAM settings in Curseforge (guide)
    • I recommend between 4,000mb and 10,000mb depending on your hardware. Always leave at least 2gb for your operating system to use!
  4. Click play on the MMA Craft modpack. If it crashes, try changing your RAM settings, or Googling the error. If you can't solve it, join the discord and message the support channel.
  5. Go to the multiplayer tab and click "Add Server" at the bottom.
  6. Name the server whatever you like, and insert this in the IP address bar:

modded.mmaminecraft.org

The default settings that I pre-loaded are on the low end of things. Turn up distant horizons and turn on shaders if you want better graphics. Turn off distant horizons and remove the resource packs if your laptop is screaming for help.
